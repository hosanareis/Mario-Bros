<h2> 📌 Funcionalidades: </h2><br>


⚡ **Formulário de contato interativo:** Ao clicar no botão “Entre em contato”, um formulário exibido em formato de janela (modal). As informações enviadas são encaminhadas diretamente para o e-mail do prestador de serviço.

📩 **Envio de mensagens por formulário:** O formulário está integrado a um serviço online que permite receber as mensagens enviadas pelos clientes diretamente no e-mail.

💬 **Contato direto pelo WhatsApp:** Ao clicar no ícone do WhatsApp, o usuário é redirecionado automaticamente para o WhatsApp da empresa, já com uma mensagem pronta para iniciar o atendimento.
